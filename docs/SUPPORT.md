# Support

## Hermes Agent vs this repository

- **[Hermes Agent](https://github.com/NousResearch/hermes-agent)** is the upstream agent runtime and tooling ecosystem.
- **This repository** ([hermes-control-hub](https://github.com/Daniel-Parke/hermes-control-hub)) is the **Control Hub** web dashboard and APIs that operate on local data and Hermes installs.

Questions about Hermes behaviour, gateways, or agent prompts are often best directed at Hermes documentation or community channels upstream. Questions about the Control Hub UI, REST routes, `CH_DATA_DIR`, or this repo’s CI belong here.

## Getting help with Control Hub

1. **Documentation** — Start at the [documentation index](README.md), especially [CONTROL_HUB.md](CONTROL_HUB.md) and [DEPLOY.md](DEPLOY.md).
2. **Bug reports and features** — Open a [GitHub Issue](https://github.com/Daniel-Parke/hermes-control-hub/issues) with a clear repro or use case.
3. **Contributing** — See [CONTRIBUTING.md](CONTRIBUTING.md) and [TESTING.md](TESTING.md) for workflow, tests, and Playwright notes.
